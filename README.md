# 프로젝트 소개
"천만 유튜버가 되자"는 꿈을 주제로 만든 모바일 세로형(16:9) 텍스트 어드벤처 & 시뮬레이션 게임입니다. 평범한 미소녀 여고생이 유튜버가 되어 
1000만 구독자를 달성하는 여정을 그립니다. 
플레이어는 10대 후반의 미소녀 캐릭터를 조종하며, 구독자를 키우는 과정에서 다양한 선택지와 도전에 직면합니다.
각 선택은 구독자수, 인기도, 스트레스 수치에 영향을 미칩니다 .
